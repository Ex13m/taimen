# ПОДРОБНАЯ ИНСТРУКЦИЯ: ключи и площадки для Тайменя

Три сервиса: **Anthropic** (мозг), **GitHub** (код + память), **Netlify** (хостинг).
Всё делается один раз, минут за 15.

---

## 1. Anthropic — ключ мозга (обязательно)

Standalone-Таймень работает по **API-биллингу** — это отдельные деньги, НЕ подписка Claude.
Каждый разговор = платные токены. Мозг Тайменя — Claude Fable 5 ($10/$50 за млн
токенов ввода/вывода), свита — Opus 4.8 ($5/$25), суб-агенты — Sonnet 5 ($3/$15).
С нашими лимитами обычный ответ Тайменя — до ~10–12 центов; spend-cap обязателен.

1. Зайди на **https://console.anthropic.com** (аккаунт тот же, что для Claude, или новый).
2. Слева **Billing** → **Buy credits** → пополни на $5–10 для начала.
3. **⚠️ Сразу поставь spend-cap**: Billing → **Spend limits** → поставь месячный
   потолок (например $10). Это твой предохранитель — даже если что-то пойдёт не так,
   больше этой суммы не сгорит.
4. Слева **API Keys** → **Create Key** → имя `taimen` → **Copy**.
   Ключ вида `sk-ant-api03-...` показывается ОДИН раз — сохрани в надёжное место.
5. Никуда в код не вставляй — только в ENV Netlify (шаг 3.4).

Доп. защита уже в коде: max_tokens ≤ 1000, ≥2 сек между запросами,
дневной потолок 200 обращений (меняется переменной `DAILY_LIMIT`).

## 2. GitHub — токен для памяти (по желанию, но рекомендую)

Без него память Тайменя живёт только в браузере (localStorage).
С ним — заметки и скормленные знания коммитятся в ветку `taimen-memory`
этого же репо и доступны с любого устройства.

1. GitHub → аватарка → **Settings** → внизу слева **Developer settings**.
2. **Personal access tokens** → **Fine-grained tokens** → **Generate new token**.
3. Заполни:
   - **Token name**: `taimen-memory`
   - **Expiration**: 90 дней или год (потом просто сделаешь новый)
   - **Resource owner**: `Ex13m`
   - **Repository access**: **Only select repositories** → выбери `taimen`
   - **Permissions** → **Repository permissions** → найди **Contents** → **Read and write**.
     Больше НИЧЕГО не включай.
4. **Generate token** → скопируй (`github_pat_...`) — тоже показывается один раз.
5. Вставишь в ENV Netlify как `GITHUB_TOKEN` (шаг 3.4).

Почему это безопасно: токен умеет только читать/писать файлы одного репо `taimen`,
живёт в ENV Netlify (не в коде), и память пишется в отдельную ветку,
не трогая код и не запуская пересборки сайта.

## 3. Netlify — сайт (обязательно)

1. **https://app.netlify.com** → войти через GitHub (кнопка **Log in with GitHub**) —
   заодно даст доступ к репозиториям.
2. **Add new site** → **Import an existing project** → **GitHub** → выбери `Ex13m/taimen`.
   Если репо не видно: **Configure the Netlify app on GitHub** → дать доступ к `taimen`.
3. Настройки сборки НЕ трогай (они уже в `netlify.toml`: publish `.`,
   functions `netlify/functions`). Branch to deploy: `main`. → **Deploy site**.
4. **Ключи**: Site configuration → **Environment variables** → **Add a variable**:

   | Key | Value |
   |---|---|
   | `ANTHROPIC_API_KEY` | `sk-ant-api03-...` из шага 1 |
   | `GITHUB_TOKEN` | `github_pat_...` из шага 2 (если делал) |
   | `DAILY_LIMIT` | по желанию, дефолт `200` |
   | `ACCESS_KEY` | по желанию: пароль, чтобы чужие не тратили твои токены. Придумай слово; на каждом своём устройстве один раз напиши в чат Тайменю: `пароль ТВОЁСЛОВО` |
   | `DAILY_COST_LIMIT` | по желанию, дефолт `5` (долларов в день на мозг; жёсткая отсечка на 110%) |
   | `SCOUT_DAILY_COST` | по желанию, дефолт `0.5` (долларов в день на авторазведку новостей) |
   | `OPENROUTER_API_KEY` | по желанию: запасной мозг, БЕСПЛАТНЫЕ модели (шаг 1b) |
   | `OPENROUTER_MODEL` | по желанию, дефолт `deepseek/deepseek-chat-v3-0324:free` |
   | `REPLICATE_API_TOKEN` | по желанию: второй запасной мозг, копейки за ответ (шаг 1b) |
   | `REPLICATE_MODEL` | по желанию, дефолт `meta/meta-llama-3-70b-instruct` |

## 1b. Запасной мозг — бесплатные модели (по желанию, проще всего с телефона)

Если ключ Anthropic не подключён / не подошёл / API недоступен — Таймень
сам переключается на запасной мозг. Два варианта, хватит одного:

**А. OpenRouter (рекомендую — по-настоящему бесплатно).**
**https://openrouter.ai** — агрегатор сотен моделей; у многих есть версия
с пометкой `:free` — $0 за токены (DeepSeek, Llama, Qwen, Mistral…), есть только
дневной лимит запросов. Карта не нужна.
1. openrouter.ai → **Sign in** (можно через Google/GitHub, удобно с телефона).
2. Меню → **Keys** → **Create Key** → скопируй (`sk-or-...`).
3. Netlify → Site configuration → Environment variables →
   `OPENROUTER_API_KEY` = ключ → **Trigger deploy**.
4. Дефолтная модель — DeepSeek V3 (free), по-русски говорит достойно. Сменить:
   переменная `OPENROUTER_MODEL` (любой ID с openrouter.ai/models, ищи фильтром FREE).
   Если поставишь платную (без `:free`) — её траты тоже считаются в дневной бюджет
   (консервативно $3/$15 за млн; уточнить — переменная `OPENROUTER_PRICE`, напр. `0.27,1.1`).

**Б. Replicate (почти бесплатно).**
**https://replicate.com** — открытые модели с оплатой за факт: Llama 3 70B
~$0.65/$2.75 за млн токенов (в ~15 раз дешевле Fable 5), обычный ответ — доли
цента; новому аккаунту дают стартовый кредит. Токен: аватарка → **API tokens** →
скопируй `r8_...` → в Netlify как `REPLICATE_API_TOKEN`.

Порядок: сначала пробуется OpenRouter, потом Replicate. В датчике расхода
запасная модель видна как `openrouter/…` или `replicate/…`. Честно: открытые
модели проще Claude — это страховка, а не замена.

## 3b. GitHub Actions — ночной мастер (самоулучшение кодом)

Чтобы Таймень сам воплощал пожелания из чата («хочу, чтобы ты умел…»):

Два способа оплаты мастера — выбери один (можно оба, подписка в приоритете):

**А. Подпиской Claude Max (дешевле — API-кредиты не тратятся).**
На компьютере с Claude Code (`npm i -g @anthropic-ai/claude-code`, вход по подписке)
выполни `claude setup-token`, скопируй токен →
GitHub → репо `taimen` → **Settings → Secrets and variables → Actions** →
**New repository secret**: Name `CLAUDE_CODE_OAUTH_TOKEN`, Value — токен.

**Б. API-ключом (жжёт кредиты, ~$2–5 за ночь с пожеланием).**
Там же секрет: Name `ANTHROPIC_API_KEY`, Value — ключ из console.anthropic.com.
2. Там же → Settings → **General** → Pull Requests → включи **Allow auto-merge**
   (тогда зелёные PR ночного мастера вливаются сами; иначе будут ждать твоей кнопки).
3. Всё: каждую ночь в 03:17 UTC мастер берёт первое пожелание из книги, пишет код,
   гоняет тесты и открывает PR. Запустить вручную: вкладка **Actions** → night-master
   → **Run workflow**.

5. После добавления переменных: **Deploys** → **Trigger deploy** → **Deploy site**
   (функции подхватывают ENV при деплое).
6. Имя сайта: Site configuration → **Change site name** → например `taimen-orb`
   → адрес `https://taimen-orb.netlify.app`.

HTTPS у Netlify из коробки — это важно: микрофон в браузере работает только по HTTPS.

## 4. Проверка (2 минуты)

1. Открой сайт → жизнь начнётся сама: заставка, новости версии, взрывное почкование.
2. Напиши «привет» → пришёл ответ голосом и текстом → мозг подключён ✓.
3. Скажи что-нибудь через 🎙 (разреши микрофон) → распозналось ✓.
4. Чип «память» справа сверху показывает **git** → память подключена ✓
   (если «локальная» — проверь `GITHUB_TOKEN` и передеплой).
5. На телефоне: открой тот же адрес в Chrome → всё то же самое.

## Если что-то не так

- **«Мозг не подключён»** — нет `ANTHROPIC_API_KEY` в ENV или не передеплоил.
- **«Ключ не подошёл»** — ключ скопирован не полностью / отозван.
- **Ответ «Таймень отдыхает до завтра»** — дневной лимит; подними `DAILY_LIMIT`.
- **Микрофон молчит** — проверь, что адрес https и разрешение выдано (замочек в адресной строке).
- **Память «локальная», хотя токен есть** — у токена нет права Contents RW именно на `taimen`.
