# СТРАТЕГИЯ «ГАЛАКТИКА» — TAIMEN как самособирающая цивилизация-ОС

Смена парадигмы (14.07.2026). Раньше: орб + свита агентов. Теперь: **живая
галактика-операционная-система**. Каждый агент — не «шарик», а **планета** со
своей цивилизацией, которая решает поставленную задачу. Галактика растёт сама,
осваивает новые планеты, развивается как цивилизация — и одновременно
играбельна как игра. Заменяет [SPEC-2.0](SPEC-2.0.md) в части модели агентов;
инженерные этапы (память, CI, автономия) из SPEC остаются в силе.

---

## 1. Метафора, которая всё меняет

| Было (свита) | Стало (галактика) |
|---|---|
| орб-помощник | **планета** с цивилизацией |
| активировать агента | **прилететь на планету**, спуститься к населению |
| агент отвечает | цивилизация планеты **решает задачу** её жителями |
| новый субагент | **колонизация**: осваивается новая планета |
| память орба | **летопись цивилизации** планеты |
| расход токенов | **ресурс** (энергия/масса), который планета тратит на труд |
| ночной мастер | **терраформирование**: галактика перестраивает саму себя |
| хозяин | **Наблюдатель** — ты, сидишь над галактикой и внутри неё |

Три масштаба наблюдения (они же три режима камеры):
1. **Галактика сверху** — карта колоний, кто чем занят, потоки ресурсов и
   «посольств» (ракеты = обмен данными между планетами).
2. **Рядом с планетой** — орбита, техно-кольца, спутники-подзадачи.
3. **Вблизи, на поверхности** — видно население: цивилизация в этот момент
   решает задачу агента (огни городов = активные вычисления, стройка = работа).

## 2. Масса галактики — чем она «весит»

Галактике нужна масса, вокруг которой всё держится. Предлагаю:
**ЯДРО — «Таймень-сингулярность»**, спящая масса глубин. Её вес = накопленная
память + знания + решённые задачи. Чем больше цивилизация прожила и запомнила —
тем тяжелее ядро, тем шире и стабильнее орбиты планет (визуально: орбиты
разъезжаются, ядро ярче пульсирует). Это делает **прогресс физически видимым**:
поработали — галактика буквально стала массивнее и красивее.

Формула-ощущение (не физика, а UX): `масса = f(знания, решённые_задачи, дни_жизни)`;
масса → радиус орбит, яркость ядра, число фоновых звёзд-«достижений».

## 3b. ТРЕТИЙ ПУТЬ — ось Прогрессора (главная миссия)

Прогрессор — полунезависимая планета по Стругацким: двигает развитие **мягким,
соразмерным вмешательством**. Его ось — **Третий Путь**, ответ на вопрос
«кем мы останемся, выходя в космос»:

- **Путь 1 — экспансия/покорение.** Звёздная империя, флоты, человек-доминанта.
  Космос как ресурс. — Отвергаем.
- **Путь 2 — трансценденция/растворение.** Постгуманизм, слияние с ИИ, отказ от
  телесности. Человек перестаёт быть человеком. — Отвергаем.
- **Путь 3 (наш).** Выйти на космический масштаб, **оставшись человеком** — этика,
  любопытство, внутренняя мера. Не покорять и не раствориться, а **повзрослеть**.
  Стругацкианский нерв: вопрос не «как далеко шагнём», а «кем при этом останемся».

Наследие Стругацких: прогрессорство, Полдень, цена вмешательства, ответственность
зрелого разума перед младшими. Вопрос нравственный, не технологический: что значит
быть **достойным космоса**. Дальняя цель Прогрессора — когда появятся ресурсы,
продвигать Третий Путь и проект **«1 доллар на бессмертие»** силами общества,
у реальной планеты Земля.

Банк — казна цивилизации: балансы всех счетов, доходы/расходы, ресурс каждой
планеты, предупреждение о перерасходе. Финансовая опора, без которой миссию
Прогрессора не потянуть.

«Осваивает новые планеты» = когда возникает новый класс задач, галактика
**колонизирует** новую планету (рождается новый агент), с собственным биомом,
который выбирается по назначению из этой палитры или изобретается.

## 4. Честная 3D-графика — дорожная карта (главная боль)

Сейчас поверхность «плоская», не видно живого 3D-дрожания. Три ступени:

**Ступень A — живой шейдер (сейчас, в коде).** Настоящее смещение поверхности:
техно-линии (широта/долгота-сетка планеты) дышат и деформируют силуэт;
рельеф из fbm-шума реально гонит рябь по поверхности (не только цвет —
смещение нормали и границы SDF). У каждого биома — свой узор линий (см. §3).
Это первый честный 3D-эффект «вибрации линиями», о котором речь.

**Ступень B — спецэффекты вселенной.** Атмосферное рассеивание (планета светится
ободом-небом), кольца, спутники-подзадачи на своих орбитах, аккреционный диск
ядра, звёздная пыль с параллаксом, «посольские» ракеты с подписью (уже есть).

**Ступень C — реальные 3D-меши (когда подключим художника).** Image-to-3D
(Tripo / Higgsfield generate_3d / SAM-3D): по сгенерированной картинке планеты
получаем настоящий GLB-меш с текстурой; грузим в сцену как объект (Three.js
или свой WebGL-лоадер). Тогда планета — не метабол, а реальная модель с
городами. Требует: генератор картинок (этап 5 SPEC — художник) + меш-конвертер
+ загрузчик GLB. Сеть сессии сейчас блокирует загрузку в Higgsfield — делаем,
когда художник поедет через свой канал (Replicate/собственный прокси).

## 5. Умная стратегия агентов = дипломатия цивилизаций

Раз агент — это планета-цивилизация, поведение агентов становится богаче
простого «ответь»:

- **Компетенция = юрисдикция планеты.** Задача приходит в галактику → ядро
  решает, чья это планета (маршрутизация по теме), и «отправляет посольство»
  (видимая ракета) к нужной планете. Ответ = делегация цивилизации.
- **Кооперация планет.** Сложная задача дробится: Стратег раскладывает на шаги
  (маршруты), Аналитик считает, Писарь оформляет, Хранитель фиксирует итог —
  и это видно как цепочка ракет между планетами в реальном времени.
- **Автономная жизнь.** Между задачами планеты «живут»: Шпион шлёт разведданные
  (scout), Иммортал крутит отсчёт, ядро копит массу. Наблюдатель видит движение,
  даже когда не спрашивает.
- **Колонизация по потребности.** Появился新 класс задач (например «рисуй»,
  «собери PDF») → галактика колонизирует новую планету-мастерскую с художником
  и печатней. Это прямой путь к «мастерской файлов» из SPEC (любые файлы на
  вход, PDF с картинками на выход = продукция планеты-мастерской).
- **Экономика ресурса.** Токены = масса/энергия. У каждой планеты свой бюджет;
  дорогая работа = ярче горит, тратит массу; отсечка перерасхода (110%) = планета
  «уходит на восстановление». Наблюдатель видит, кто сколько «съедает».

## 6. Игровой слой (одновременно игра)

- **Цель игры = рост цивилизации**, а инструмент роста — решённые реальные задачи
  Наблюдателя. Решил задачу → галактика тяжелеет, планета развивается, открывается
  новый биом/эффект. Польза и игра — одно и то же действие.
- **Прогресс виден**: масса ядра, число колоний, «эры» (мажорные версии = смены
  эпохи цивилизации с анимацией), достижения-звёзды на фоне.
- **Голосовое управление флотом**: «покажи Аналитика» = облёт к планете;
  «обновись/освой новое» = колонизация/терраформирование (этап 4 SPEC —
  голосовой запуск мастера, после кода просит перегруз = «эра сменилась,
  нырнуть в новую галактику?»).

## 7. Что делаем по шагам (сшивка с инженерной SPEC)

1. **Сейчас**: живой шейдер (ступень A) + этот документ + иллюстрированная
   «Книга галактики» с фото Наблюдателя. Метафора внедряется в тексты и «что
   нового», имена планет пока прежние (Стратег/Аналитик…), безопасно.
2. **Этап 0 SPEC** (предохранители: смоук в CI, пароль, санитайзинг) — обязателен
   до самопроектирования.
3. **Этап 1 SPEC** (память-крепость) → становится «летописью цивилизации».
4. **Этап 5 SPEC** (мастерская + художник) → планета-мастерская + ступень C
   (реальные 3D-меши планет по картинкам).
5. **Полный ребрендинг в «планеты»** (имена, UI, режимы камеры «сверху/рядом/
   поверхность») — крупным PR, руками, после смоука в CI.

## 8. Честно о рисках/границах

- Реальные 3D-меши (ступень C) — самая тяжёлая часть: нужен генератор картинок
  без сетевых блоков + GLB-загрузчик; это отдельный крупный этап, не «на ночь».
- Ребрендинг в планеты трогает много кода (ENTITIES, подписи, инфо-окна, голос,
  штурвал) — делать одним обдуманным PR под смоук-CI, иначе риск белого экрана.
- Игровой слой добавляет состояние (масса, эры, достижения) — оно тоже должно
  жить в «летописи» (память-крепость этапа 1), чтобы прогресс не пропадал.
