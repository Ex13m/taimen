# ТАЙМЕНЬ 🜁 · v1.0.0

Голографический дух-помощник: настоящее 3D (raymarching-метаболы) — свита
почкуется из главного шара; голос в обе стороны (Web Speech API), скармливание
материалов, самообучающаяся память с сохранением в git, PWA «на домашний экран».
Один HTML-файл, vanilla JS, ноль npm-зависимостей фронта.

## Запуск за 3 шага

1. **Ключи** (см. «Где вставить ключи» ниже) — минимум `ANTHROPIC_API_KEY`.
2. **Локально**: `netlify dev` в корне репо → открыть http://localhost:8888
   (или любой статик-сервер, но тогда `/api/*` работать не будет — чат скажет об этом честно).
3. **Прод**: пуш в `main` → Netlify собирает сам (`netlify.toml` уже настроен).

## Где вставить ключи

Все секреты — ТОЛЬКО в ENV Netlify (Site configuration → Environment variables).
В коде и git ключей нет и быть не должно.

| Переменная | Обязательна | Что делает |
|---|---|---|
| `ANTHROPIC_API_KEY` | да | мозг Тайменя (chat.js → api.anthropic.com) |
| `DAILY_LIMIT` | нет (дефолт 200) | потолок обращений к мозгу в сутки |
| `GITHUB_TOKEN` | нет | постоянная память: fine-grained PAT c правом Contents RW на этот репо; без него память живёт в localStorage браузера |
| `GITHUB_REPO` | нет (дефолт `Ex13m/taimen`) | куда писать память |
| `MEMORY_BRANCH` | нет (дефолт `taimen-memory`) | ветка памяти (отдельная, чтобы не триггерить деплои) |
| `ACCESS_KEY` | нет | пароль на сайт: без него чат и память отвечают 401; вводится в чате командой «пароль СЛОВО» |
| `ELEVENLABS_API_KEY` | нет | красивый голос (этап E4.2, пока заглушка) |

⚠️ Не забудь поставить **spend-cap в консоли Anthropic** — standalone работает по API-биллингу.

## Устройство

```
index.html               фронт: орб (WebGL-шейдер), чат, голос, свита, знания, память
netlify/functions/
  chat.js                прокси-мозг: валидация, лимиты (2с/IP + дневной), max_tokens<=1000
  memory.js              память: GitHub contents API, ветка taimen-memory
  tts.js                 заглушка ElevenLabs (E4.2)
netlify.toml             /api/* -> functions
```

Контракт мозга: `POST /api/chat {messages, system, model?, max_tokens?} → {text}`.
Модели: `claude-sonnet-4-6` (Таймень) и `claude-haiku-4-5` (свита и дистилляция памяти).

## Как жить с Тайменем

- «Запуск» → говори (🎙) или пиши. Ответ звучит голосом (🔊 переключает).
- «✦ знания» → вставить текст или бросить .txt/.md в окно — Таймень опирается на них.
- Свита справа: Стратег, Аналитик, Писарь, Хранитель. Таймень сам зовёт нужного —
  орб подсвечивается; тап по орбу передаёт ему слово (дешёвая модель), тап по большому орбу — возврат.
- Память: каждые 6 реплик Таймень сам выписывает факты; чип «память» справа сверху,
  список и удаление — в панели знаний.
